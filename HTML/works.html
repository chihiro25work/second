制作概要・目的

転職活動で使用するポートフォリオ。
制作物や自分自身の強みなどを伝える。書類では伝わりにくい人柄についてもより深い理解を促し、一緒に働くイメージを持っていただく。

ターゲット

企業の採用担当者の方々

写真でインパクトを与えることと、温かみや活力、明るさなどのイメージを人々に抱かせる暖色を使用することで親しみやすさを演出ました。

鳴川ちひろのポートフォリオサイト②

created date 2025.04

制作概要・目的

転職活動で使用するポートフォリオ。
制作物や自分自身の強みなどを伝える。書類では伝わりにくい人柄についてもより深い理解を促し、一緒に働くイメージを持っていただく。

ターゲット

企業の採用担当者の方々

情報設計

作品や自分について興味を持っていただけるよう情報整理とわかりやすいシンプルなデザインを心掛けました。ターゲットが知りたい情報を簡潔に伝えることを最優先に考え、一番知りたい項目は「制作物」であると想定し、一番初めに配置しました。

デザイン

シンプルで一般的な構成になっているので、キービジュアルに動きをつけたり、コンテンツの背景は幼少期の写真が覗いたりすることにより、変化を取り入れました。

制作期間

デザイン:1日 HTML5,CSS3コーディング:3日間

使用ツール・言語

Figma / Photoshop / HTML5 / CSS3 / JavaScript

キルトサロンダックリンの生徒募集バナー

created date 2025.04

制作概要・目的

キルトサロンダックリンのキルト教室の生徒募集を促すバナー制作

ターゲット

徳島県内でパッチワークキルトに興味がある方々

情報設計

教室に通うことでどのような技術を学ぶことができ、どのような作品を作成することができるのかを明確に記載しました。
バナーのみでは伝えきれない情報が多いため、直接問い合わせることができるよう店舗直通の電話番号を記載しました。

デザイン

手作りの温かみを連想させる、パステルカラーやアースカラーの茶色を使用しました。
背景にはダックリンで販売されているキットの見本を使用しました。

制作期間

1日

使用ツール

Photoshop
